2. Концепт и фантазия

Тото заперт в подвешенной клетке с навесными замками над огнём. Игрок жмёт SPIN — на барабанах выпадают 3 scatter-символа корзины, запускается бонус «Pick a Basket». Игрок тапает корзины одну за другой: в каждой открытой корзине лежит золотой ключ и кредитный приз/множитель. Ключ вылетает из корзины, подлетает к клетке и срывает один из навесных замков, после чего пламя становится ниже. Когда все три замка сорваны, клетка распахивается, Тото свободен, а собранные призы складываются в большой умноженный выигрыш.

_Эмоциональный крючок: спасение животного под нарастающей угрозой (огонь) + классическая жадность pick-bonus («что в следующей корзине?»). Каждый пик даёт двойную награду — приз + ослабление угрозы._

**Референс ключевого экрана (визуальное направление):**

_Композиция: верх — сцена-угроза (клетка с Тото над огнём), низ — слот-барабаны + HUD. Конкретные символы на скрине — пример визуального направления, не финальный paytable._

6. Механика бонуса «Pick a Basket»

_После триггер-спина барабаны схлопываются/затемняются, поверх или вместо них появляется сетка корзин. Игрок видит подсказку «Pick a basket to free Toto»._

| **Параметр**              | **Значение**                                                                                                                                                                                                                                                                           |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **_Количество корзин_**   | **_6 корзин на сетке (2×3 или 3×2). Игрок выбирает 3 из 6._**                                                                                                                                                                                                                          |
| **_Что внутри корзин_**   | **_В КАЖДОЙ корзине лежит золотой ключ + приз (кредит или множитель). Ключ — обязательный компонент, обеспечивает сюжетную связь с клеткой. Призы: small/mid/big credit, множители ×2/×3/×5. Конкретные значения — TBD._**                                                             |
| **_Связь с клеткой_**     | **_После открытия корзины ключ вылетает по дуге к клетке, вставляется в один из навесных замков и проворачивается. Замок щёлкает и падает. Пламя ниже на 1 уровень. Порядок снятия замков — по индексу пика (1-й пик_** → **_1-й замок и т.д.), визуально замки определены заранее._** |
| **_Финальное умножение_** | **_Сумма кредитов из 3 пиков × финальный множитель (или один из множителей в корзинах применяется ко всей сумме)._**                                                                                                                                                                   |
| **_Детерминизм_**         | **_Содержимое каждой корзины зашито под индекс пика. Порядок выбора игрока влияет на оптику, не на сумму (1-й пик всегда = small/mid, 3-й = big — для нарастающего drama)._**                                                                                                          |
| **_Подсказки_**           | **_Pulse на доступных корзинах, лёгкое свечение под курсором при наведении/ховере._**                                                                                                                                                                                                  |

9. Правила взаимодействия

- **Единый ввод.** в spin-фазе активна только кнопка SPIN. В bonus-фазе активны только корзины (6 шт.). Остальной экран не реагирует.
- **Тапы.** 1 тап на SPIN + 3 тапа на корзины = 4 активных тапа. Это максимум engagement для slot-плеебла батча. Снижать не рекомендуется.
- **Детерминизм.** результат spin-а (3 scatter) и содержимое каждой из 6 корзин зашиты. Игрок может выбрать любые 3 из 6, но порядок наполнения детерминирован по индексу пика (1-й = small, 2-й = mid+multiplier, 3-й = big).
- **Idle / подсказки.** если игрок не жмёт SPIN N сек — пульс кнопки усиливается + стрелка-подсказка; через 2 цикла — авто-спин. В бонус-фазе — pulse на корзинах, через N сек авто-пик случайной корзины.
- **CTA.** после payout — авто-переход на end-card. На end-card любой тап = редирект в стор.
