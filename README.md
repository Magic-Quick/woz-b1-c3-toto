# woz-b1-c3-toto

Сценарий слота «Save Toto» (WOZ B1‑C3) представляет игру‑вишер с механикой 3‑скаттер‑триггер, переходящей в бонус «Pick‑a‑Basket», где игрок выбирает 3 из 6 корзин, получая ключи и призы, освобождая Тото из клетки, снижая уровень огня и в конце получая умноженный выигрыш; документ включает детали формата, UI‑элементов, логики, таймингов, ассетов и требований к согласованию с клиентом.
